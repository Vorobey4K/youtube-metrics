# YouTube Metrics Reports

Проект для анализа метрик YouTube-видео и генерации отчётов на основе CSV-файлов.

Скрипт загружает данные из файлов, фильтрует их по заданным правилам и выводит результат в виде таблицы. Проект удобно расширять для добавления новых типов отчётов.

Скриншоты работы находятся в папке screenshots.
