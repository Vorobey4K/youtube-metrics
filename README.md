# YouTube Metrics Reports

Проект для анализа метрик YouTube-видео и генерации отчётов на основе CSV-файлов.

Скрипт загружает данные из файлов, фильтрует их по заданным правилам и выводит результат в виде таблицы.  
Проект легко расширяется за счёт добавления новых отчётов.

Примеры входных данных находятся в папке `data/`.  
Примеры вывода отчётов представлены в папке `screenshots/`.
